How It Works

1. Install

Add the BlockItAll extension to Chrome or Firefox. It takes 10 seconds.

2. Subscribe

Choose which block lists you want — porn bots, engagement farmers, or all of them.

3. Block

Hit "Sync & Block" and the extension blocks every account on your lists. It runs in the background — you can keep browsing.

Your Account Is Safe

We know asking you to install a browser extension is a big trust ask. Here's exactly how BlockItAll works — no hand-waving.

How is this different from giving you my password?

When you log into Twitter, your browser stores a small piece of data called a session token in its cookies. This token tells Twitter "this browser is logged in as @you" — it's what keeps you signed in between page loads.

The BlockItAll extension reads this token locally from your browser's cookies — the same way Twitter's own website reads it. The token never leaves your computer. It's never sent to our servers or anywhere else. The extension uses it to send block requests directly from your browser to Twitter, exactly as if you were clicking "Block" on each profile yourself.

Unlike your password, a session token can't be used to change your password, email, or security settings. And if you ever want to revoke it, just log out of Twitter — the token becomes invalid immediately.

How to verify the code yourself

BlockItAll is 100% open source. Every line of code the extension runs on your browser is public on GitHub.

  • Read the source code on GitHub
  • Search for fetch( and XMLHttpRequest in the extension code — you'll see that the only outbound requests go to twitter.com/x.com (for blocking) and our public block list API (for downloading the list of handles to block)
  • The extension never sends your session token, cookies, or any auth data anywhere except directly to Twitter
